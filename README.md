# erettsegi-browser

- **Framework**: [Next.js](https://nextjs.org/)
- **Deployment**: [Vercel](https://vercel.com)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)

## Running Locally

```bash
git clone https://github.com/skidoodle/erettsegi-browser
cd erettsegi-browser
yarn install
yarn dev
```
